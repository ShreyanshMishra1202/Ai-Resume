## Quickstart

### Backend

```bash
cd server
npm install
cp .env.example .env
npm run dev
```

### Frontend

```bash
cd client
npm install
cp .env.example .env
npm run dev
```

## Environment

- Server: set `MONGODB_URI`, `JWT_SECRET`
- Client: set `VITE_API_URL`

```bash
cd server
npm install
npm run dev
```
